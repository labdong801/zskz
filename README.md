

## 中山市抢口罩脚本

### 环境
- Python 3.6
- beautifulsoup4 4.8.2
- requests 2.22.0
- pillow 7.0.0
- pycrypto 2.6.1
- pycryptodome 3.8.2

### 使用方法

（1）安装依赖：

````
pip install -r requirements.txt
````

（2）编辑main.py。填写手机号码。

（3）开抢
````
python main.py
````

### 注意事项

验证码输入后2分钟内有效，请勿太早执行。
